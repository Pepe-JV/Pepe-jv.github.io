Weather App

Aplicación meteorológica moderna con React

React 18 Framer Motion Visual Crossing API Responsive Design

Descripción del Proyecto

Una aplicación web moderna y responsiva para consultar el pronóstico del tiempo en tiempo real. Permite a los usuarios buscar cualquier ubicación en el mundo y ver información meteorológica detallada con una interfaz elegante y animaciones fluidas.

Características Principales

Búsqueda de Ubicaciones
Busca el clima de cualquier ciudad o ubicación en el mundo con resultados instantáneos.
Información Completa
Temperatura actual, sensación térmica, humedad, viento y probabilidad de lluvia.
Pronóstico por Horas
Visualiza el pronóstico de las últimas 12 horas y las próximas 12 horas (24h total).
Geolocalización
Detecta automáticamente tu ubicación al cargar para mostrar el clima local.
Animaciones Suaves
Transiciones elegantes con Framer Motion para una experiencia visual atractiva.
Diseño Responsive
Se adapta perfectamente a móviles, tablets y escritorio con un diseño fluido.

Funcionalidades Implementadas

Requisitos Básicos

  • Campo de entrada para búsqueda de ubicaciones

Cómo Funciona

La aplicación utiliza la API de Visual Crossing Weather para obtener datos meteorológicos en tiempo real. Los usuarios pueden buscar cualquier ubicación o usar la geolocalización del navegador para obtener el clima local.

Ejemplo de Uso

Flujo de Uso 
1. Abre la aplicación
2. Permite el acceso a la ubicación (opcional)
3. O escribe una ciudad: "Madrid", "Nueva York", "Tokio"
4. Presiona "Buscar" o Enter
5. Visualiza el clima actual y el pronóstico de 24 horas
6. Usa "Actualizar" para refrescar los datos

API Key Configurada

La aplicación ya incluye una API key funcional de Visual Crossing Weather con 1000 consultas gratuitas por día.

Nota: Si deseas usar tu propia API key, puedes obtenerla gratuitamente en Visual Crossing Weather API y reemplazarla en el archivo app.js.

Bash
